## 一、Hurst 指数的估计方法

### 1.1 R/S（Rescaled Range）分析法

#### 历史背景

R/S 分析法由英国水文学家 Harold Edwin Hurst 于 1951 年在研究尼罗河水位长期变化规律时首次提出，后经 Mandelbrot 和 Wallis (1969) 推广为分析时间序列长程相关性的标准工具。该方法是最古老、最经典的 Hurst 指数估计方法。

#### 算法步骤

给定一个长度为 N 的时间序列 {x_1, x_2, ..., x_N}：

**第一步：计算均值**

    m = (1/N) * sum(x_i, i=1..N)

**第二步：构建均值调整序列**

    Y_t = x_t - m,   t = 1, 2, ..., N

**第三步：计算累积离差序列（Cumulative Deviates）**

    Z_t = sum(Y_i, i=1..t),   t = 1, 2, ..., N

**第四步：计算极差（Range）**

    R(n) = max(Z_1, Z_2, ..., Z_n) - min(Z_1, Z_2, ..., Z_n)

**第五步：计算标准差**

    S(n) = sqrt( (1/n) * sum((x_i - m)^2, i=1..n) )

**第六步：计算重标极差**

    (R/S)_n = R(n) / S(n)

对不同的子序列长度 n 重复上述计算，对相同长度的多个子序列取平均值。

**第七步：估计 Hurst 指数**

在 log-log 坐标中，绘制 log(R/S) 对 log(n) 的图。Hurst 指数 H 由以下渐近关系的斜率确定：

    E[R(n)/S(n)] = C * n^H,  n -> infinity

即：

    log(R/S) = log(C) + H * log(n)

对 log-log 图做线性回归，斜率即为 H 的估计值。

#### 优点
- 算法直观、实现简单
- 历史悠久，理论基础扎实
- 对于平稳时间序列有较好的表现
- 不需要假设信号的具体分布

#### 缺点与已知精度问题
- **对非平稳性敏感**：当时间序列中存在趋势（线性或非线性）时，R/S 法倾向于检测出虚假的长程相关性，因为趋势会导致极差 R 增长过快，从而高估 H 值
- **有限样本偏差**：在有限样本条件下，R/S 分析系统性地高估 Hurst 指数。Anis & Lloyd (1976) 和 Hamed (2007) 对此进行了修正研究
- **短序列表现差**：当序列较短时，均值的估计可能严重偏离真实值，导致累积离差序列出现系统性偏差
- **不能处理非平稳数据**：这是 R/S 法最根本的局限——真实世界（包括音乐）的数据通常是非平稳的

#### 参考文献
- Hurst, H.E. (1951). "Long-term storage capacity of reservoirs." *Transactions of the American Society of Civil Engineers*, 116, 770-808.
- Mandelbrot, B.B., & Wallis, J.R. (1969). "Robustness of the rescaled range R/S in the measurement of noncyclic long run statistical dependence." *Water Resources Research*, 5(5), 967-988.
- Hamed, K.H. (2007). "Improved finite-sample Hurst exponent estimates using rescaled range analysis." *Water Resources Research*, 43(7).

### 1.2 DFA（Detrended Fluctuation Analysis，去趋势波动分析）

#### 提出者与原始论文

DFA 方法由 **C.-K. Peng, S.V. Buldyrev, S. Havlin, M. Simons, H.E. Stanley, A.L. Goldberger** 于 1994 年在分析 DNA 核苷酸序列的"马赛克组织"特性时首次提出。

**原始论文**：
- Peng, C.-K., Buldyrev, S.V., Havlin, S., Simons, M., Stanley, H.E., & Goldberger, A.L. (1994). "Mosaic organization of DNA nucleotide sequences." *Physical Review E*, 49(2), 1685-1689. （截至 2022 年被引用超过 3000 次）

**后续方法论文**：
- Peng, C.-K., Havlin, S., Stanley, H.E., & Goldberger, A.L. (1995). "Quantification of scaling exponents and crossover phenomena in nonstationary heartbeat time series." *Chaos*, 5(1), 82-87.

#### 算法步骤

给定一个长度为 N 的时间序列 {x_1, x_2, ..., x_N}：

**第一步：计算均值**

    <x> = (1/N) * sum(x_t, t=1..N)

**第二步：构建积分序列（Profile）**

将原始时间序列积分（累积求和），形成"轮廓线"：

    X(k) = sum( (x_i - <x>), i=1..k ),   k = 1, 2, ..., N

**第三步：分段**

将积分后的序列 X(k) 划分为 N_s = floor(N/n) 个等长为 n 的不重叠片段。为充分利用数据，通常从序列两端分别划分，得到 2*N_s 个片段。

**第四步：局部去趋势**

在每个长度为 n 的片段内，用最小二乘法拟合一个多项式趋势线 Y_n(k)。
- DFA-1：使用一阶多项式（线性拟合），可去除线性趋势
- DFA-2：使用二阶多项式，可去除二次趋势
- DFA-k：使用 k 阶多项式，可去除 k 阶趋势

**第五步：计算波动函数 F(n)**

在每个片段中计算去趋势后的均方根波动：

    F(n) = sqrt( (1/N) * sum( [X(k) - Y_n(k)]^2, k=1..N ) )

更精确地说，对每个片段 v 计算局部波动：

    F(n,v) = sqrt( (1/n) * sum( [X((v-1)*n + j) - Y_n^v(j)]^2, j=1..n ) )

然后对所有片段取平均：

    F(n) = sqrt( (1/(2*N_s)) * sum( F(n,v)^2, v=1..2*N_s ) )

**第六步：多尺度分析**

对不同的片段长度 n（通常取 n = 4, 5, 6, ..., N/4），重复第三至第五步，得到 F(n) 关于 n 的函数关系。

**第七步：确定标度指数 alpha**

在 log-log 坐标中绘制 log(F(n)) 对 log(n) 的图。如果数据具有幂律标度特性，则图上呈现线性关系：

    F(n) ~ n^alpha

直线的斜率即为 DFA 标度指数 alpha。

#### 与 R/S 方法的比较优势

| 比较维度 | R/S 分析 | DFA |
|---------|---------|-----|
| **处理非平稳性** | 不能去除趋势，易产生虚假长程相关性 | 通过局部多项式拟合去除趋势，可处理非平稳数据 |
| **短序列表现** | 有限样本偏差较大 | 对短序列（低至约 2048 个数据点）仍有较好精度 |
| **抗噪能力** | 对外部趋势和周期性干扰敏感 | 可通过提高去趋势阶数来消除不同类型的非平稳性 |
| **应用范围** | 限于平稳或接近平稳的序列 | 专门设计用于非平稳序列，这正是真实音乐数据的特点 |
| **标度指数精度** | 倾向于系统性高估 H | 估计偏差较小，但标准差可能略高于 R/S |

**DFA 的核心优势**：它允许检测嵌入在看似非平稳时间序列中的内在自相似性，同时避免外部趋势引起的虚假自相似性检测。正如 Peng 等人所指出的，DFA "permits the detection of intrinsic self-similarity embedded in a seemingly nonstationary time series."

**DFA 的局限性**：DFA 也只能处理有限类型的非平稳性。简单的正弦周期性或单调全局趋势仍然可能影响 DFA 的估计结果。

#### Jennings et al. 将 DFA 应用于音乐分析

**关键论文**：
- Jennings, H.D., Ivanov, P.Ch., Martins, A.M., da Silva, P.C., & Viswanathan, G.M. (2004). "Variance fluctuations in nonstationary time series: a comparative study of music genres." *Physica A*, 336(3-4), 585-594.

Jennings 等人的贡献是将 DFA 方法从生物医学信号分析领域引入音乐信号分析。他们使用"去趋势方差波动指数"（detrended variance fluctuation exponent）对不同音乐流派进行分类。主要发现：

- DFA 指数能有效估计音乐的"可舞性"（danceability），区分具有不同节奏结构的流派
- 较低的 DFA 指数值对应较高的可舞性，说明更容易与节奏元素同步
- 电子技术舞曲（techno）的平均 DFA 指数显著低于古典乐曲的 DFA 指数
- 该方法揭示了音乐数据在不同时间尺度上的相关性结构

后续研究中，Streich & Herrera 使用 7750 首音轨和 172 个语义标签的数据集，进一步验证了 DFA 指数与音乐语义特征之间的相关性。

## 二、分形维度在音乐中的应用

### 2.1 盒计数维度（Box-Counting Dimension）在旋律序列中的应用

#### 基本原理

盒计数维度（又称 Minkowski-Bouligand 维度）是计算分形维度最简单、最广泛使用的方法。其核心思想是：用边长为 epsilon 的"盒子"覆盖目标集合，统计所需的最少盒子数 N(epsilon)，然后通过以下关系确定分形维度：

    D_box = lim(epsilon->0) [ log(N(epsilon)) / log(1/epsilon) ]

实际计算中，对不同的 epsilon 值统计 N(epsilon)，在 log-log 坐标中绘制 log(N(epsilon)) 对 log(1/epsilon)，斜率即为盒计数维度的估计值。

#### 应用于旋律序列的方法

将旋律转化为可用盒计数分析的图形表示，主要有以下几种方式：

1. **旋律轮廓法（Melodic Contour）**：将每个音符的音高值按时间顺序绘制为点，然后用线段连接相邻的音符，形成"旋律轮廓线"。对此曲线应用盒计数法。轮廓线的分形维度反映旋律的"粗糙程度"——维度越高，旋律起伏越复杂。

2. **音程散点图法（Interval Scatter Plot）**：基于相邻音符之间的音程构建散点图，然后计算其分形维度。但研究表明此方法的分形维度与音乐复杂度的相关性较弱。

3. **音高时间序列法（Pitch Time Series）**：直接将音高值视为时间序列，计算其分形特征。

#### 关键数值

根据 Niklasson & Niklasson (2020, arXiv:2004.02612) 的研究：

- **古典音乐**的盒计数维度较高，例如巴赫作品约 **D ≈ 1.58**
- **民间音乐**的盒计数维度较低，约 **D ≈ 1.36**
- 音频信号的盒计数维度通常在 **D ≈ 1.6 - 1.7** 范围
- 旋律轮廓线的分形维度范围为 **1 < D < 2**

#### 方法论上的注意事项

Niklasson & Niklasson 的研究同时指出了盒计数法的重要局限：
- 盒计数法和 DFA 法给出的分形维度可能存在显著差异
- 盒计数法在维度接近 2 时精度较低
- 将离散音高通过线段连接形成旋律轮廓可能引入偏差
- 有研究者质疑盒计数法在音乐中的适用性：音高数据不一定满足幂律分布的前提条件
- 相比之下，DFA 方法对短时间序列（2048 个数据点以内）表现更好

### 2.2 分形维度与旋律复杂度的关系

分形维度 D 与旋律复杂度之间的理论关系：

- **D 接近 1**：旋律轮廓接近光滑曲线，意味着音高变化平缓、可预测性强——听感上"单调"或"平滑"（对应棕色噪声特征）
- **D 接近 2**：旋律轮廓接近填满整个平面的路径，意味着音高变化剧烈、跳跃频繁——听感上"混乱"或"嘈杂"（对应白噪声特征）
- **D 约 1.5**：介于两者之间，兼具结构性和变化性——这正是 1/f 噪声（粉色噪声）对应的复杂度水平，也是被认为最"悦耳"的旋律特征

通过 Hurst 指数 H 与分形维度的关系：

    D = 2 - H

可以在两个指标之间互相转换：
- H = 0.5（白噪声）=> D = 1.5
- H = 0.8（粉色噪声典型值）=> D = 1.2
- H = 1.0（棕色噪声极限）=> D = 1.0

### 2.3 Mandelbrot 分形理论与音乐自相似性

#### 分形与自相似性

Benoit Mandelbrot 于 1975 年创造了"分形"（fractal）一词，用于描述在不同尺度上都呈现相似结构的几何形状——"打碎后的每一部分都是整体的缩小模型"。

音乐中的自相似性表现在：
- **时间尺度不变性**：在不同时间尺度上观察音乐信号的统计特征保持不变。如果放大或缩小时间轴，信号波动的统计特性不会改变
- **结构层级嵌套**：音乐的动机（motif）在乐句、段落、乐章等不同层次上以变形的方式反复出现，形成"大结构呼应小结构"的层级关系
- **幂律频谱**：音乐信号的功率谱密度遵循 1/f^beta 的幂律关系，这正是分形过程的频域特征

#### 自仿射性（Self-Affinity）vs 自相似性（Self-Similarity）

Mandelbrot 强调音乐中的分形特性更准确地说是**自仿射**（self-affine）而非严格的自相似（self-similar）。自仿射意味着在 x 方向（时间）和 y 方向（音高）的缩放比例不同。频谱指数 beta 是一个全局度量，是自仿射指数而非分形维度。

#### Mandelbrot 的相关著作
- Mandelbrot, B.B. (1999). *Multifractals and 1/f Noise: Wild Self-Affinity in Physics (1963-1976)*. Springer.
- Mandelbrot, B.B. (2002). *Gaussian Self-Affinity and Fractals: Globality, The Earth, 1/f Noise, and R/S*. Springer.
- Hsu, K.J., & Hsu, A. (1991). "Self-similarity of the '1/f noise' called music." *Proceedings of the National Academy of Sciences*, 88(8), 3507-3509.

## 三、长程相关性（Long-Range Correlation）

### 3.1 音乐中长程相关性的物理意义

#### 什么是长程相关性

长程相关性（也称长程依赖、长程记忆）指的是时间序列中相距较远的值之间仍然存在统计相关性。与短程相关（自相关函数指数衰减）不同，长程相关的自相关函数以**幂律形式**缓慢衰减：

    C(tau) ~ tau^(-gamma),   0 < gamma < 1

这意味着远处的事件仍然对当前有影响，只是影响随距离逐渐减弱但永远不会消失。

#### 长程相关性如何让旋律听起来"自然"

1/f 噪声（粉色噪声）恰好处于**可预测性与意外性的最佳平衡点**：

- **白噪声**（无相关性）：每个音符与前后完全无关。听感上过于随机跳跃，大脑无法建立预期模式——令人"疲惫"和"不安"
- **棕色噪声**（过强相关性）：音高变化过于平滑，近乎完全可预测。听感上过于单调，缺乏信息量——令人"无聊"
- **1/f 噪声**（适度长程相关性）：大脑的预测"大多正确但偶尔出错"。旋律的大致走向可以预期，但细节上有惊喜——这正是音乐让人"愉悦"和"自然"的关键

用信息论的语言说：1/f 噪声提供了**最大化信息传递效率**的编码方式。大脑神经网络本身也呈现 1/f 动态特性，因此对 1/f 结构的信号具有天然的亲和力。

Voss & Clarke (1978) 通过实验验证了这一点：用 1/f 噪声源控制频率和持续时间生成的旋律听起来"令人愉悦"（sounded pleasing），而白噪声源生成的旋律"过于随机"（too random），1/f^2 噪声源生成的旋律"过于相关"（too correlated）。

#### 自然界的 1/f 普遍性

1/f 波动被发现广泛存在于自然界和人类活动中，包括：心跳间隔、脑电波、河流流量、地震活动、股票价格波动、语言中的词频分布等。音乐作为人类创造活动的产物，自然地继承了这种普遍的统计规律。

### 3.2 不同噪声类型的长程相关性差异

| 噪声类型 | 功率谱 S(f) | beta 值 | 自相关特性 | Hurst 指数 H | DFA 指数 alpha | 听感特征 |
|---------|-----------|--------|----------|------------|--------------|---------|
| **白噪声** | S(f) = const | beta = 0 | 无相关性（delta 函数） | H = 0.5 | alpha = 0.5 | 完全随机，跳跃无序 |
| **粉色噪声 (1/f)** | S(f) ~ 1/f | beta = 1 | 幂律衰减，长程相关 | H = 0.5* (fGn) | alpha = 1.0 | 自然、平衡、悦耳 |
| **棕色噪声 (1/f^2)** | S(f) ~ 1/f^2 | beta = 2 | 累积性强相关 | H = 0.5* (fBm) | alpha = 1.5 | 过于平滑，缺乏变化 |

> *注意：Hurst 指数的解读取决于分析的是增量序列（fractional Gaussian noise, fGn）还是累积序列（fractional Brownian motion, fBm），详见第 3.3 节。

### 3.3 自相关函数的衰减模式与噪声类型的理论关系

#### 自相关函数与功率谱的关系

根据 Wiener-Khinchin 定理，功率谱密度 S(f) 是自相关函数 C(tau) 的傅里叶变换：

    S(f) = integral( C(tau) * exp(-2*pi*i*f*tau) dtau )

因此，功率谱的幂律形式 S(f) ~ 1/f^beta 直接对应于自相关函数的幂律衰减形式 C(tau) ~ tau^(-gamma)。

#### 三者之间的精确关系

功率谱指数 beta、自相关衰减指数 gamma 和 DFA 标度指数 alpha 之间的关系：

    beta = 2*alpha - 1
    gamma = 2 - 2*alpha
    alpha = (beta + 1) / 2 = (2 - gamma) / 2

这些关系将三个独立的测量方式统一在同一个理论框架下。

#### 不同噪声类型的衰减模式

**白噪声 (beta = 0)**：
- 自相关函数：C(tau) = delta(tau)，只在 tau = 0 时非零
- 含义：完全无记忆，每个值与其他值独立
- gamma 不适用（非幂律衰减）

**粉色噪声 (beta = 1)**：
- 自相关函数：C(tau) ~ tau^(-gamma)，gamma = 0（对数衰减）
- 含义：相关性衰减极慢，几乎保持恒定——最强的长程记忆
- 这是 1/f 噪声的标志性特征

**棕色噪声 (beta = 2)**：
- 作为白噪声的累积和（随机游走/布朗运动），不是平稳过程
- 其增量序列（白噪声）的自相关为 delta 函数
- 累积序列本身不具有传统意义上的自相关函数

## 四、关键数值参考

### 4.1 真实音乐的 Hurst 指数和频谱指数典型值

#### Voss & Clarke (1975, 1978) 的经典测量

Voss & Clarke 分析了多种音乐录音（巴赫勃兰登堡协奏曲、爵士乐、摇滚乐、广播音乐等）的音频功率波动，发现：
- 功率谱密度普遍近似 1/f，即 **beta 约 1**
- 频率范围覆盖约 3 个数量级，下至 5 x 10^(-4) Hz
- 这一结果对多种音乐风格和时代都适用

#### Su & Wu (2007) "Music Walk" 的测量

Su & Wu 将乐谱中的音符序列转换为一维随机游走（"音乐漫步"），用 Hurst 指数和 Fourier 频谱分析进行量化测量：

- 分析的音乐曲目包括 Gavotte、Le Cygne、Ave Maria 等的小提琴谱
- 音高序列的 Hurst 指数范围：**H = 0.22 ~ 0.40**
- 所有值都小于 0.5，表现为**反持续性**（anti-persistent），即旋律音高具有"上行后倾向下行、下行后倾向上行"的回归特性
- 频谱呈现 1/f^b 型特征，在不同时间尺度上可能有两个不同的 b 值

> 注意：H < 0.5 对应的是音高的**增量序列**（即音程序列）的特性，反映的是旋律"避免单方向持续运动"的倾向——这正是好的旋律既有方向感又会转折的原因。

#### Levitin, Chordia & Menon (2012) 的节奏频谱分析

Levitin 等人分析了 558 首西方古典音乐的 1,788 个乐章的节奏频谱：

**总体统计**：
- 节奏频谱指数中位数：**beta = 0.78 (SD = 0.35)**
- 范围约 **0.5 ~ 1.0**
- "与 1 无显著差异"——即普遍服从 1/f 幂律

**作曲家差异**：
- **贝多芬**：beta 值最大（最具可预测性的节奏）
- **莫扎特**：beta 值最小（最少可预测性的节奏）
- **海顿**：介于莫扎特和贝多芬之间，与两者都有显著差异
- **蒙特威尔第与乔普林**（Joplin）：节奏 beta 分布几乎完全相同
- 具体范围：**0.48 <= beta <= 1.05**

**体裁差异**：
- 最可预测（最大 beta）：交响曲和弦乐四重奏
- 最少预测（最小 beta）：玛祖卡舞曲和拉格泰姆

**重要发现**：同一作曲家的音高频谱（pitch spectrum）几乎相同，但节奏频谱（rhythm spectrum）呈现显著差异——作曲家通过系统性地操纵（有意识或无意识地）1/f 节奏中的可预测性来赋予作品独特的风格标识。

### 4.2 beta 值与 Hurst 指数的理论关系

#### 分数高斯噪声（Fractional Gaussian Noise, fGn）

fGn 是一种平稳的长程相关过程，是分数布朗运动的增量序列。

    beta = 2H - 1        (beta 范围: [-1, 1])
    H = (beta + 1) / 2
    alpha_DFA = H         (alpha 范围: [0, 1])

| H 值 | beta 值 | alpha 值 | 含义 |
|------|---------|---------|------|
| 0 | -1 | 0 | 强反相关 |
| 0.5 | 0 | 0.5 | 白噪声（无相关） |
| 0.75 | 0.5 | 0.75 | 中等正相关 |
| 1.0 | 1 | 1.0 | 1/f 噪声（最强长程相关 fGn） |

#### 分数布朗运动（Fractional Brownian Motion, fBm）

fBm 是 fGn 的累积和（积分），是非平稳的自仿射过程。

    beta = 2H + 1        (beta 范围: [1, 3])
    H = (beta - 1) / 2
    alpha_DFA = H + 1     (alpha 范围: [1, 2])

| H 值 | beta 值 | alpha 值 | 含义 |
|------|---------|---------|------|
| 0 | 1 | 1 | 1/f 噪声 |
| 0.5 | 2 | 1.5 | 标准布朗运动 |
| 1.0 | 3 | 2 | 最平滑的 fBm |

#### 统一关系式

无论是 fGn 还是 fBm，都满足以下通用关系：

    alpha = (beta + 1) / 2
    beta = 2*alpha - 1
    gamma = 2 - 2*alpha    （gamma 为自相关函数幂律衰减指数）

其中 alpha 是 DFA 标度指数，beta 是功率谱指数，gamma 是自相关衰减指数。

> 关键判断：alpha < 1 时数据类似 fGn（平稳），alpha > 1 时数据类似 fBm（非平稳）。分界点 alpha = 1 对应 1/f 噪声。

### 4.3 DFA 标度指数 alpha 与 Hurst 指数 H 的关系

这两个指数的关系取决于信号的性质：

**情况一：信号是 fGn 类型（平稳增量过程），alpha < 1**

    alpha = H

DFA 标度指数直接等于 Hurst 指数。

**情况二：信号是 fBm 类型（非平稳累积过程），alpha > 1**

    alpha = H + 1
    即 H = alpha - 1

**实际应用中的注意事项**：

在音乐分析中，研究者可能分析的是：
- **音高序列本身**（累积量，类似 fBm）：此时 alpha = H + 1
- **音高的差分序列/音程序列**（增量，类似 fGn）：此时 alpha = H

因此，在引用文献中的 H 值时，必须弄清楚原始研究分析的是哪种序列，否则会产生系统性的误解。

### 4.4 常用噪声类型的参数总结表

| 参数 | 白噪声 | 粉色噪声 (1/f) | 棕色噪声 (1/f^2) |
|------|--------|--------------|----------------|
| 功率谱指数 beta | 0 | 1 | 2 |
| DFA 指数 alpha | 0.5 | 1.0 | 1.5 |
| fGn 的 H | 0.5 | 1.0 | - |
| fBm 的 H | - | 0 | 0.5 |
| 分形维度 D = 2-H (fBm) | - | 2.0 | 1.5 |
| 自相关衰减指数 gamma | 不适用 (delta) | 0 (对数衰减) | 不适用 (非平稳) |
| 信号特征 | 完全随机 | 长程相关 | 随机游走 |
| 旋律听感 | 跳跃、混乱 | 自然、平衡 | 平滑、单调 |
